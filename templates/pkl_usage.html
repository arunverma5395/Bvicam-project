Your Guide to prediction using Yourmodel.pkl files.

  1. Download the .pkl file.
  2. load the element to be predicted in a variable using Value = prediction
  3. Open the terminal and type pip install joblib . (Wait for it to download.)
  4. In your program navigate to the folder containing the Yourmodel.pkl file.
  5. load the pickle file to a variable using myModel = joblib.load('Yourmodel.pkl')
  6. Predict the outcome using the myModel.predict(Value)

and VOILA!!
